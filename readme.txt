Projet réalisé au lycée avec Aloszko Martin, Aubry Joshua et Vallée Mathieu.

Il s'agit d'un jeu de combat codé en python avec la librairie Pygame.

Pour lancer, il faut soit :
- lancer Le '.exe'
- ou lancer le '.py'


Si vous avez un problème de résolution d'écran ou de touches, vous pouvez les modifier
directement en jeu dans le menu options ou alors dans le fichier 'Option.txt'.
